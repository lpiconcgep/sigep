\# SIGEP - Sistema de Gestión de Postgrados



¡Bienvenido al repositorio de \*\*SIGEP\*\*! Este proyecto tiene como objetivo principal gestionar la información de los programas de postgrado perteneciente a la Coordinación General de Estudios de Postgrado de la Universidad de Los Andes CGEP-ULA.



\## 📋 Descripción



SIGEP es una aplicación web desarrollada en \*\*PHP\*\* para administrar los datos de postgrados, Estudiantes, listado de postgrados creados oficialmente desde CNU, y la gestión de las inscripciones según los expedientes que lleguen a la dependencia.
